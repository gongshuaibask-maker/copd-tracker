## 截图制作方法

### 方法 1：真机截图（推荐）

```bash
# 启动 Expo 开发服务器
npx expo start

# iOS 模拟器截图
# 1. 打开 Simulator
# 2. File → Save Screenshot (⌘S)

# Android 模拟器截图
# 1. 打开 Android Emulator
# 2. 点击相机图标 → Save Screenshot
```

### 方法 2：Web 截图

```bash
# 启动 Web 版本
npx expo start --web

# 使用浏览器开发者工具
# 1. F12 → 设备模拟 (Ctrl+Shift+M)
# 2. 选择 iPhone 14 Pro Max
# 3. 截图保存
```

### 方法 3：自动化工具

```bash
# 使用 fastlane 自动截图
fastlane screengrab

# 或在 Playwright 中编写截图脚本
```
