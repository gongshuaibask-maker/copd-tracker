## 描述（中文简体）

### 简短描述（80 字内）

慢阻肺（COPD）患者的全方位自我管理工具。记录肺功能、症状评分、血氧等 11 大类健康数据，趋势图表自动生成，OCR 智能识别检查报告。数据 100% 本地存储，隐私无忧。

### 完整描述（4000 字内）

**慢阻肺自我管理助手**是一款专为慢阻肺（COPD）患者设计的健康自我管理应用。无论您是刚确诊还是长期管理，本应用都能帮助您系统化地记录和追踪健康数据，让病情变化一目了然。

**🏥 12 大功能模块，全方位覆盖**
- 🫁 **肺功能检查**：记录 FEV₁、FVC、FEV₁/FVC、PEF 等 20+ 项指标，支持肺活量判定与舒张试验分析，自动提取报告结论
- 🔬 **气道炎症**：追踪 FeNO、血 EOS、痰 EOS、痰 NEUT 等气道炎症指标
- 📋 **症状评分**：CAT 8 题量表 + mMRC 呼吸困难分级，国际标准科学评估，含急性加重史记录
- 🏃 **运动耐力**：6 分钟步行试验记录，Borg 呼吸困难评分，前后血氧/心率对比
- 💓 **氧合呼吸**：日常 SpO₂、心率、呼吸频率监测，异常值自动预警
- ⚖️ **营养体重**：体重与 BMI 追踪，结合身高自动计算营养状况
- 🚨 **急性加重**：发作事件记录，年度频次统计，GOLD 2026 风险等级评估（低/中/高）
- 💊 **用药疗效**：急救药使用频次、吸入操作规范评分 1-5
- 🏥 **合并症**：PAP、T值、FBG、HbA1c、血脂四项等检查记录
- 🚭 **戒烟管理**：每日吸烟量记录，FTND 尼古丁依赖量表评估
- 😴 **睡眠监测**：夜间 SpO₂、ODI、T90、最低心率追踪
- 🏋️ **肺康复**：运动类型、时长、Borg 评分追踪

**📸 新增特色功能**
- **照片档案**：每个模块独立照片库，检查报告拍照保存，永不丢失
- **健康筛查**：PHQ-2 抑郁筛查、GAD-2 焦虑筛查、SARC-F 肌少症筛查
- **行动计划**：绿/黄/红三区 COPD 自我管理方案，与医生协作制定
- **疫苗接种记录**：流感、肺炎球菌、COVID-19、RSV 等疫苗全程追踪
- **提醒系统**：每日血氧测量、月度 CAT 评分、复查提醒
- **报告结论提取**：OCR 后自动提取诊断结论文字和检查日期

**🔒 隐私优先，数据只属于你**
- 所有数据使用 SQLite 存储在您的设备本地
- 不上传云端、不收集个人信息、不追踪行为
- 无需网络权限即可使用全部功能
- 支持一键 JSON 数据导出备份

**🤖 OCR 智能识别**
- 粘贴或拍照检查报告，自动识别肺功能等 20+ 项指标
- 识别结果可手动修正，确认后自动填入表单
- 支持多家医院常见报告格式

**📊 趋势可视化**
- 自动生成折线趋势图
- 支持 3 月 / 6 月 / 1 年 / 全部时间范围切换
- 各模块独立图表分析

**🔔 智能提醒**
- 每日指脉氧测量提醒
- 月度 CAT 评分提醒
- 复查时间提醒

**📱 特性一览**
- ✅ 完全免费，无广告、无订阅、无内购
- ✅ 纯本地化，无需注册账号
- ✅ 支持 iOS + Android + Web
- ✅ 开源项目，持续更新
- ✅ 界面简洁清晰，适合中老年用户

**⚠️ 医疗免责声明**
本应用仅供个人健康管理参考，不构成医疗诊断或治疗建议。如有身体不适，请及时就医。所有健康数据的解读应在专业医师指导下进行。

## 描述（English）

### Short Description

A comprehensive self-management tool for COPD patients. Track lung function, symptoms, SpO₂, and 8 more health categories. Auto-generated trend charts, OCR report recognition. 100% offline, privacy-first.

### Full Description

**COPD Self-Management** is a health tracking app designed specifically for patients with Chronic Obstructive Pulmonary Disease (COPD). Whether you're newly diagnosed or managing a long-term condition, this app helps you systematically record and monitor your health data.

**🏥 12 Modules, Full Coverage**
- 🫁 Pulmonary Function: FEV₁, FVC, FEV₁/FVC, PEF and 20+ indicators, report conclusion extraction
- 🔬 Airway Inflammation: FeNO, blood EOS, sputum EOS tracking
- 📋 Symptom Assessment: CAT-8 questionnaire + mMRC dyspnea scale
- 🏃 Exercise Tolerance: 6-minute walk test with Borg score
- 💓 Oxygen & Respiration: Daily SpO₂, heart rate, respiratory rate
- ⚖️ Nutrition & Weight: BMI tracking with automated calculation
- 🚨 Exacerbation: Episode logging with GOLD 2026 risk assessment
- 💊 Medication: Rescue inhaler frequency, inhalation technique scoring
- 🏥 Comorbidities: PAP, T-score, FBG, HbA1c, blood lipids
- 🚭 Smoking Cessation: Daily log + FTND dependency assessment
- 😴 Sleep Monitoring: Nocturnal SpO₂, ODI, T90 tracking
- 🏋️ Pulmonary Rehab: Exercise type, duration, Borg score

**🔒 Privacy First**
- All data stored locally on your device using SQLite
- No cloud upload, no data collection, no tracking
- Works completely offline
- One-tap JSON data export

**📊 Key Features**
- ✓ 100% Free - No ads, no subscriptions, no in-app purchases
- ✓ Fully offline - No account required
- ✓ OCR text recognition for medical reports
- ✓ Auto-generated trend charts with time range filter
- ✓ Smart reminders for daily measurements
- ✓ Clean, accessible UI designed for all ages
- ✓ Open source

## 上架流程

```bash
# iOS App Store
eas build --platform ios --profile production    # 构建
eas submit --platform ios --profile production    # 提交

# Android Google Play
eas build --platform android --profile production # 构建 AAB
eas submit --platform android --profile production # 提交
```

> **注意**：提交前需在 `eas.json` 中填入 Apple ID、App Store Connect App ID、Apple Team ID。
