## 🗿 Hacker News — Show HN

### 标题
```
Show HN: COPD Self-Management – Free, offline-first health tracker, 12 modules
```

### 正文
```
I built this because a family member with COPD needed a simple way to track multiple health metrics without dealing with complex apps or privacy concerns.

The entire app runs locally using SQLite — zero data ever leaves the device.

Key features that might interest HN:
- React Native + Expo 54 + TypeScript
- Fully offline (SQLite on device)
- OCR text recognition for medical reports
- 12 modular tracking components
- GOLD 2026 compliant risk assessment
- Clean medical UI with visual design tokens

App: https://copd-selfcare.netlify.app
GitHub: https://github.com/gongshuai888/copd-tracker

Technical stack: Expo 54, TypeScript strict mode, SQLite, React Native Paper, react-native-chart-kit

Would love feedback on the architecture, UX, or anything else!
```
